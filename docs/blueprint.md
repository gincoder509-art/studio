# **App Name**: Gincoder Portfolio Pro

## Core Features:

- Portfolio Showcase: Display a selection of your best web development projects (restaurant, boutique, barber, clinique, ecole, formation, eglise, onglerie, particulier), with images and descriptions.
- Flyer Integration: Incorporate the multi-services flyer, ensuring it is prominently displayed and easily accessible on the website.
- Contact Buttons: Create direct contact buttons (WhatsApp, Facebook, TikTok, call) to facilitate easy communication with potential clients.
- Image gallery generator: AI-powered tool that creates images relevant to a project, by analysing the content and logo.

## Style Guidelines:

- Primary color: Dark, saturated teal (#008080) to inspire trust and authority, referencing the Gincoder logo's green while avoiding pure green to signal a professional edge.
- Background color: Very dark desaturated teal (#0A2525), creating a professional dark theme.
- Accent color: Desaturated blue (#468A99) to complement the primary and offer visual contrast.
- Headline font: 'Space Grotesk' sans-serif for a tech-forward look, well-suited to a developer portfolio.
- Body font: 'Inter' sans-serif provides a clean and modern appearance, well-suited to readability.
- Use simple, modern icons to represent different sections and functionalities. Prefer icons in a 'flat' design.
- Clean, modern layout with a focus on showcasing projects and providing easy access to contact information.
- Subtle animations on hover for project images to provide a polished, interactive user experience.
